## 1. 产品概述

项目风险管控系统是一款面向项目管理团队的专业风险管控工具，旨在从进度偏差、资源冲突、需求变更、外部依赖等多维度持续识别项目风险，通过概率×影响的量化评估模型对风险进行精准定级，并支持预防措施和应急响应预案的制定与追踪。系统聚焦趋势变化与根因分析，帮助团队将风险管控从被动救火转变为主动防控。

- **目标用户**：项目经理、项目群经理、PMO、技术负责人、风险管理员
- **核心价值**：将风险管控从静态快照升级为动态趋势追踪，从表象描述深入到根因剖析，实现真正的前置化风险管理

## 2. 核心功能

### 2.1 用户角色

| 角色 | 权限说明 |
|------|----------|
| 项目经理 | 创建/编辑风险、制定响应预案、更新风险状态、查看所有分析报表 |
| 团队成员 | 查看风险登记册、上报风险、更新负责的风险措施状态 |
| PMO/高管 | 查看跨项目风险总览、趋势报表、风险热力图 |

### 2.2 功能模块

1. **风险总览仪表盘**：全局风险态势感知，风险热力图、趋势曲线、关键指标卡片、预警通知
2. **风险登记册**：完整的风险清单管理，多维筛选排序，风险详情卡片，状态流转追踪
3. **风险识别与评估**：多维度风险录入，概率×影响矩阵评估，风险等级自动计算，根因标记
4. **趋势分析中心**：风险趋势折线图、维度对比雷达图、风险演进时间线、异常波动检测
5. **响应预案管理**：预防措施与应急预案制定，措施执行追踪，预案效果回溯评估

## 3. 核心流程

**风险管控主流程**：项目经理或团队成员从各维度识别潜在风险 → 对风险进行概率×影响的量化评估 → 根据评估结果制定预防措施和应急预案 → 持续跟踪风险状态变化 → 定期更新风险登记册 → 通过趋势分析发现异常 → 调整响应策略

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题，主色采用钢蓝色（#1B3A5C）搭配琥珀色（#F59E0B）作为警示强调色，营造专业、冷静、可信赖的氛围
- **辅助色**：风险等级色彩体系 — 极高（#EF4444 红色）、高（#F97316 橙色）、中（#EAB308 黄色）、低（#22C55E 绿色）
- **按钮风格**：圆角微弧（6px），扁平化设计，关键操作用琥珀色突出
- **字体**：标题使用 DM Sans（粗体醒目），正文使用 Noto Sans SC（中文友好），数据数字使用 JetBrains Mono（等宽清晰）
- **布局风格**：左侧固定导航栏 + 顶部面包屑 + 主内容区域，卡片式模块布局
- **图标风格**：线性图标（Phosphor Icons），2px 描边，与整体专业风格一致

### 4.3 响应式设计

- 桌面优先设计，核心内容区最小宽度 1200px
- 平板端：导航栏折叠为图标模式，卡片由4列变为2列
- 移动端：导航栏收为底部标签栏，图表简化为关键数据

### 4.4 动效设计

- 页面加载：模块依次淡入上移（stagger 100ms）
- 风险等级变化：颜色渐变过渡（300ms ease）
- 热力图气泡：悬停放大+tooltip
- 趋势图：数据点绘制动画（逐点显现）
- 预警通知：从右侧滑入，自动消失
- 详情面板：右侧滑出（300ms ease-out）
